# nextjs+ts

A Next.js + TypeScript template (App Router), aligned in complexity with vite-react+ts.

## Scripts

- dev: Start dev server
- build: Type check then build
- start: Start production server
- lint / lint:fix: ESLint checks

## Structure

- app/: App Router entry with layout and a demo page
- public/: Static assets (logo.svg, favicon.svg)
- .eslintrc.js / .prettierrc.js: Lint & format settings
- tsconfig.json: TS config

## Usage

1. cd project-templates/frontend/nextjs+ts
2. npm install
3. npm run dev
