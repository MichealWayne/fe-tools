# 典型 Nodejs 服务端项目构建

- `koa2`: koa2 + typescript
- `fastify`(beta): fastify + typescript
- `nestjs`: nestjs + typescript

## 使用方法

1. 选择模板目录，复制到新项目
2. 安装依赖
3. 参考模板内 README 开发
