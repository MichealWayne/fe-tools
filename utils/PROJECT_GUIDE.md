# utils

前端常用 js 工具集。

## 工程说明

- `ai-utils`：ai 相关工具函数；
- `canvas-utils`：canvas 操作相关工具函数；
- `node-img-build`：nodejs 操作图片相关工具函数；
- `node-utils`：nodejs 基础常用工具函数；
- `utils`：js 常用工具函数（环境无关）；
- `web-utils`：浏览器常用工具函数

### 1.安装

使用`pnpm`

```sh
pnpm install
```

### 2.执行单测

```sh
npm run test
```

### 3.构建

```sh
npm run build
```

### 4.生成文档

```sh
npm run docs
```
