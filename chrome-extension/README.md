# FrontEnd Tools Chrome Plugin

FE-Tools 是一款面向前端开发者的 Chrome 浏览器插件，采用 Vue 3 + TypeScript + Vite 构建，支持 Chrome Manifest V3，并提供中文 / English 双语界面。

> 本目录为插件产物，可直接安装。源码见：[https://github.com/MichealWayne/fe-tools-chrome-plugin](https://github.com/MichealWayne/fe-tools-chrome-plugin)

## 安装方式

1. 下载或克隆本仓库到本地。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 打开右上角的「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择本仓库下的 `chrome-extension/` 目录。
6. 安装完成后，可在浏览器工具栏中固定 FE-Tools 插件图标，方便日常使用。

> 如果更新了本目录下的插件文件，请回到 `chrome://extensions/`，在 FE-Tools 插件卡片上点击刷新按钮重新加载。

## 使用方式

- **打开工具面板**: 点击浏览器工具栏中的 FE-Tools 图标，即可打开插件主面板。
- **搜索与快捷跳转**: 在首页输入关键词，可搜索工具库、书签，或跳转到 MDN、GitHub、NPM、CanIUse、StackOverflow 等站点。
- **地址栏快捷搜索**: 在 Chrome 地址栏输入 `fe` 后按空格或 Tab，再输入关键词，可使用插件提供的 omnibox 搜索入口。
- **右键菜单**: 在页面中可通过右键菜单使用插件提供的快捷能力，具体菜单项以当前版本展示为准。
- **页面类工具**: 页面截图、技术栈判断等能力会作用于当前活动标签页，首次使用时请按 Chrome 提示授予相关权限。
- **本地设置**: 语言、拼音搜索等偏好会保存在浏览器本地存储中，重新打开插件后仍会保留。

## 核心功能

### 1. 首页搜索

- **多平台搜索**: 支持 MDN、GitHub、NPM、CanIUse、StackOverflow、Google、百度等站点搜索。
- **工具库搜索**: 聚合远程前端工具链接，并按名称、描述、标签匹配。
- **书签搜索**: 读取浏览器书签并在输入关键词时匹配。
- **拼音搜索**: 可在设置中开启，支持中文内容的拼音和首字母匹配。
- **URL 识别**: 输入 `http` 开头的地址时，可直接进入二维码生成。
- **关键词高亮**: 搜索结果会高亮字面命中的关键词。

### 2. 设置窗口

- **语言切换**: 支持中文和英文界面。
- **拼音搜索开关**: 控制首页搜索是否启用拼音匹配。
- **偏好保存**: 设置项会写入本地存储，后续打开保持一致。

### 3. 二维码生成器

- **文本 / URL 转二维码**: 输入内容后生成二维码。
- **当前页面入口**: 可通过 URL 参数携带页面地址生成二维码。
- **SVG 下载**: 支持导出 SVG 矢量文件。
- **PNG 保存提示**: 可通过右键保存二维码图片。

### 4. 图片压缩工具

- **多格式支持**: 支持 JPG、PNG、GIF 等常见图片格式。
- **拖拽上传**: 支持点击选择和拖拽上传。
- **压缩比例**: 可设置 0-1 的压缩比例。
- **Base64 输出**: 输出压缩后图片的 Base64 编码。
- **重置图片**: 可清空当前图片重新选择。

### 5. 色值转换器

- **多格式互转**: 支持 HEX、RGB、HSB、HSL 相互转换。
- **实时预览**: 输入色值后显示颜色效果。
- **格式说明**: 页面提供 RGB、HEX、HSB、HSL 的简要说明。

### 6. API 测试工具

- **HTTP 方法**: 支持 GET、POST、PUT、DELETE、PATCH、HEAD、OPTIONS。
- **请求头编辑**: 支持添加、删除请求头。
- **请求体类型**: 支持 None、JSON、Form Data、x-www-form-urlencoded、Raw。
- **鉴权配置**: 支持无认证、Bearer Token、Basic Auth、API Key，API Key 可放入 Header 或 Query。
- **环境变量**: 支持创建环境、管理变量、导入导出环境配置。
- **响应查看**: 展示响应体、响应头、Cookies、状态码、耗时和大小。
- **历史记录**: 保存请求历史，支持复用和清空。
- **导入导出**: 支持请求配置 JSON 文件导入 / 导出。

### 7. 语言翻译

- **外部翻译入口**: 首页提供有道翻译快捷入口。

### 8. 单位换算器

- **CSS 单位互转**: 支持 px、rem、vw 换算。
- **rem 比例**: 默认 `1rem = 75px`，可自定义。
- **精度设置**: 可设置结果保留的小数位数。

### 9. Moo-CSS 查询

- **模块搜索**: 查询 Moo-CSS 模块和样式属性。
- **文档跳转**: 可跳转到相关文档页面。

### 10. 正则表达式工具

- **常用正则**: 收录前端开发常用正则表达式。
- **关键词筛选**: 按名称或分类筛选。
- **实时测试**: 输入测试字符串后验证匹配结果。

### 11. 工具方法查询

- **函数检索**: 搜索常用 JavaScript 工具函数。
- **模块索引**: 按模块列表定位工具库文档。

### 12. JSON 转换与格式化

- **JS / JSON 互转**: 支持 JavaScript 对象字面量与 JSON 双向转换。
- **格式化输出**: 支持默认、压缩、2 空格、4 空格缩进。
- **复制输出**: 一键复制转换结果。
- **错误提示**: 转换失败时显示具体错误信息。

### 13. SVG 编辑器

- **代码输入 / 上传**: 支持粘贴 SVG 代码或上传 SVG 文件。
- **在线优化**: 支持移除注释、空属性、空文本、隐藏元素、元数据、未使用命名空间，清理 ID，优化颜色值等。
- **预览与统计**: 展示 SVG 预览、原始大小、优化后大小和压缩比例。
- **复制 / 下载**: 支持复制优化结果和下载 SVG 文件。

### 14. 日期与时间戳转换

- **多格式输入**: 支持 Unix 时间戳、ISO、本地时间、自定义格式。
- **多时区输出**: 支持中国、日本、英国、中欧、美国东部、美国太平洋、UTC 等时区。
- **常用结果**: 输出秒 / 毫秒时间戳、ISO、本地时间、UTC、相对时间。
- **日期计算**: 支持按秒、分、时、日、周、月、年添加或减去时间。

### 15. Linux 命令查询

- **命令搜索**: 按命令名筛选。
- **详情查看**: 展示语法、示例和常用选项。
- **快捷命令**: 提供常用命令快速入口。

### 16. 页面截图

- **整页截图**: 自动滚动并拼接完整页面截图。
- **节点截图**: 在当前页面选择元素并截图。
- **预览与保存**: 截图后可预览、重新截图并保存到本地。

### 17. 技术栈判断

- **当前页面分析**: 分析活动标签页的技术栈特征。
- **支持类别**: 覆盖框架、构建工具和常见子库。
- **置信度展示**: 按高 / 中 / 低置信度展示命中结果。
- **证据说明**: 展示命中的脚本、DOM、全局变量、meta、页面内容等信号。
