# FrontEnd Tools Chrome Plugin

FE-Tools 是一款专为前端开发者设计的 Chrome 浏览器插件，集成了多种实用工具，旨在提高前端开发效率。插件采用 Vue3 + TypeScript 技术栈构建，支持 Chrome Manifest V3 标准，并提供中英文界面切换。

> 本目录为插件产物，可直接安装。源码见：[https://github.com/MichealWayne/fe-tools-chrome-plugin](https://github.com/MichealWayne/fe-tools-chrome-plugin)

## 核心功能

### 1. 智能搜索引擎
- **多平台搜索**: 支持在 MDN、GitHub、NPM、CanIUse、StackOverflow、Google、百度等平台快速搜索
- **书签搜索**: 自动匹配浏览器书签，快速定位收藏的网站
- **工具库搜索**: 内置前端工具库检索与聚合，统一展示搜索结果
- **URL 识别**: 输入 URL 时可直接跳转二维码生成
- **关键词高亮**: 搜索结果中关键词高亮显示，提升可读性

### 2. 二维码生成器
- **URL 转二维码**: 输入任意 URL 地址生成对应二维码
- **当前页面二维码**: 一键生成当前浏览页面的二维码
- **SVG 下载**: 支持导出 SVG 矢量图
- **实时预览**: 输入内容即时生成预览

### 3. 图片压缩工具
- **多格式支持**: 支持 JPG、PNG、GIF 等常见图片格式
- **拖拽上传**: 支持点击选择和拖拽上传两种方式
- **压缩比调节**: 自定义压缩比例 (0-1)，平衡文件大小和图片质量
- **Base64 输出**: 自动生成压缩后图片的 Base64 编码
- **即时预览**: 上传后立即显示压缩效果

### 4. 色值转换器
- **多格式互转**: 支持 HEX、RGB、HSB、HSL 四种颜色格式相互转换
- **实时预览**: 输入色值后实时显示颜色效果
- **智能校验**: 自动验证输入格式的正确性

### 5. PostMan 接口测试
- **HTTP 请求**: 支持 GET、POST、PUT、DELETE、PATCH、HEAD、OPTIONS 等请求方式
- **多种 Content-Type**: 支持 JSON、form-data、x-www-form-urlencoded、raw 等数据格式
- **请求参数**: 支持请求头、请求体编辑与环境变量替换
- **鉴权配置**: 支持 Bearer、Basic、API Key 方式
- **响应展示**: 展示响应数据、状态码、耗时与大小
- **历史记录**: 支持请求历史保存与一键复用
- **导入导出**: 支持请求配置 JSON 文件导入/导出

### 6. 语言翻译
- **文本翻译**: 输入文本自动翻译并支持快速重新翻译
- **快捷入口**: 提供外部翻译网站的快捷跳转

### 7. 单位换算器
- **像素单位转换**: 支持 px、rem、vw 等 CSS 单位互转
- **参数设置**: 可配置 rem 比例与保留小数位

### 8. Moo-CSS 查询
- **模块搜索**: 快速查询 Moo-CSS 框架的模块、方法与样式字典
- **文档集成**: 直接跳转相关文档与说明页面

### 9. 正则表达式查询
- **常用正则**: 收录前端开发中常用的正则表达式
- **实时测试**: 输入测试内容快速验证匹配结果
- **分类整理**: 按用途分类，便于快速查找

### 10. 工具方法查询
- **函数检索**: 搜索常用 JavaScript 工具函数
- **模块索引**: 按模块列表快速定位工具库文档

### 11. JSON 转换与格式化
- **JS/JSON 互转**: 支持对象字面量与 JSON 双向转换
- **格式化输出**: 支持压缩、2/4 空格缩进等格式化选项
- **复制输出**: 一键复制结果到剪贴板

### 12. SVG 编辑器
- **在线优化**: 去注释、清理属性、压缩颜色等多种优化选项
- **预览与统计**: 预览 SVG 并显示体积变化
- **复制/下载**: 支持复制优化结果与下载 SVG 文件

### 13. 日期与时间戳转换
- **多格式输入**: 支持时间戳、ISO、本地时间、自定义格式输入
- **时区切换**: 支持多时区输出转换结果
- **常用输出**: 统一展示秒/毫秒时间戳、ISO、本地、UTC、相对时间
- **日期计算**: 支持加减秒/分/时/日/周/月/年

### 14. Linux 命令查询
- **关键词筛选**: 支持命令名、描述、语法过滤
- **快捷搜索**: 提供常用命令快捷按钮
- **示例与选项**: 展示命令示例与常用参数说明

### 15. 页面截图
- **整页截图**: 自动滚动拼接全页面截图
- **节点截图**: 选择页面元素并截图保存
- **本地保存**: 支持 PNG 预览与保存到本地
